### Como instalar o Git
 
1. Acesse [git-scm.com](https://git-scm.com) e baixe o instalador para o seu sistema operacional.
2. Execute o instalador e siga as opções padrão.
3. Abra o terminal e confirme a instalação com o comando abaixo.
```bash
    git --version
```
 
4. Configure seu nome e e-mail:
```bash
    git config --global user.name "Seu Nome"
    git config --global user.email "seu@email.com"
```
 
!!! success "Pronto"
    O Git está instalado e configurado.
